# Alexey Balz - Personal Website

This is the source code for my personal GitHub Pages website at [alexey-balz.github.io](https://alexey-balz.github.io).

## Contents

- **index.html** - Main landing page
- **cv.html** - CV viewer page with PDF embed and download functionality
- **ai-collaboration-statement.html** - Statement about AI collaboration for my projects
- **style.css** - Main stylesheet
- **cv-style.css** - CV page stylesheet
- **files/** - CV PDF and other files
- **projects/** - Recent projects showcase

## Features

- Responsive design for mobile and desktop
- Embedded CV viewer using Google Docs
- Links to LinkedIn and GitHub profiles
- AI collaboration statement with verification

## Deployment

This repository is deployed automatically via GitHub Pages. Any changes pushed to the main branch will be automatically deployed to https://alexey-balz.github.io.
